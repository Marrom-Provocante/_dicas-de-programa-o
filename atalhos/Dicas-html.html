Tags HTML

  • <!DOCTYPE html> - indica a versão html
  • <html> - conteúdo do código htmtl
  • <head> - informações do cabeçalho
  • <body> - envolve todo o corpo do documento
  • <!--> - comentário
  • <title> - título da página web (na aba)
  • <meta> - informações do documento (metadados)
  • <link>- inclusão de outros documentos
  • <style> - estilização do conteúdo indicado
  • <script> - conteúdo de linguagem script
  • <p> - parágrafo
  • <a> - criação de links
  • <img> - imagem
  • <pre> - texto pré formatado (com enter, tab)
  • <pre> - texto pré formatado (com enter, tab)
  • <h1> - cabeçalho escrito (vai de h1 – h6, sendo h1 o maior tamanho de letra e h6 menor)
  • <br>- quebra de linha
  • <link rel="stylesheet" href="endereço do arquivo css">inserir seu arquivo css dentro do arquivo html(colocar codigo dentro do head

Estruturas

  • <main> - conteúdo principal do corpo
  • <header> - cabeçalho da página
  • <footer> - rodapé da página
  • <section> - define uma seção na página
  • <article> - define um artigo na página
  • <aside> - conteúdo “ao lado” (lateraliza)
  • <nav> - seção de links de navegação
  • <div> - define uma divisão na página

Formularios

  • <form> - define um formulário
  • <input> - caixa de entrada de dados
  • <textarea> - campo de texto
  • <select> - seleção de opções
  • <option> - define uma opção
  • <label> - legenda para os campos

Lista

  • <nav> - cria grupo de navegação dentro de um menu
  • <ol> - lista ordenada
  • <ul> - lista não ordenada
  • <li> - um item da lista
  • <menu> - menu com lista de itens
  • <dir> - lista de diretórios
  • <dl> - lista de descrição
  • <dt> - termo a ser descrito
  • <dd> - descrição

Formatação

  • <b> - negrito
  • <i> - itálico
  • <u> - sublinhado
  • <strong> - maior ênfase no texto em negrito
  • <em> - maior ênfase no texto em itálico
  • <sup> - deixa o texto superior a escrita
  • <sub> - deixa o texto a baixo da escrita
  • <pre> - texto pre formatado(mantem Formataçãode escrita na programação
  • <code> - reproduz linha de codigo na pagina

Adicionais

  • target=”_blank” – abre o link em outra aba do navegador
  • src=”” – é obrigatório e guarda o caminho da imagem
  • alt=”” – não é obrigatório, mas é recomendado, ele mostra a descrição da imagem quando ela não é carregada
  • <link rel="stylesheet" type="text/css" href="stylesheet.css"linkar o html ao css>
  • <script src=”nomedoarquivo”> </script>> - linkar o javascript ao html
