Dicas CSS

  • <link rel="stylesheet" href="endereço do arquivo css">inserir seu arquivo css dentro do arquivo html(colocar codigo dentro do head
  • padding - o espaço ao redor do conteúdo (ex.: ao redor do texto de um parágrafo).
  • border - a linha contínua do lado de fora do padding.
  • margin - o espaço externo a um elemento.
  • width - largura de um elemento
  • height - altura de um elemento
  • opacity - ocupacidade da imagem
  • box-shadow - cria uma sombra no objeto ou pagina
  • background-color - a cor atrás do conteúdo de um elemento e do padding.
  • color - um cor do conteúdo de um elemento (geralmente texto).
  • text-shadow - cria uma sombra no texto dentro de um elemento.
  • text-aligndisplay - defina uma maneira de divertir um elemento (não se preocupe com isso ainda).
  • /*seu comentario*/- colocar comentario dentro do css

Efeitos CSS

  • hover - permite a criação de diversos efeitos interessantes em uma página HTML
  • transition: opacity - faz uma transiçao de ocupacidade
  • transition - adiciona efeitos de transição ao modificar características visuais em um objeto
